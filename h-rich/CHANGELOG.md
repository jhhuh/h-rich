# Revision history for h-rich

## 0.1.0.0 -- 2026-01-26

* Initial implementation of core Rich functionality in Haskell.
* Added `HRich.Color` for ANSI and HTML color support.
* Added `HRich.Style` for terminal styling.
* Added `HRich.Segment` for styled text fragments.
* Added `HRich.Text` with basic BBCode-like markup support.
* Added `HRich.Console` for rich printing.
* Minimal dependencies: `base`, `text`, `containers`.
